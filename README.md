# Socioeconomic Factors That Affect Quality of Life

Group project for DATA 350 analyzing the relationship between socioeconomic factors
and quality of life outcomes using real-world data.

The analysis explores indicators such as income, education, employment, and housing
through data cleaning, visualization, and statistical methods.

## Files
- `.Rmd` file containing the full analysis and code
- Rendered `.html` report with results and visualizations

## Academic Context
Completed as a group assignment for DATA 350. Shared for educational and portfolio purposes.
